# NULINE. 極簡純淨護膚品牌官網

> 以「零添加、純植萃、可量化」為核心理念打造的護膚品牌形象網站

---

## 專案簡介

NULINE. 是一個護膚品牌的靜態形象網站，包含品牌故事、產品系列介紹、研發歷程時間軸、肌膚諮詢表單與人才招募等完整頁面內容。

---

## 使用技術

| 類別 | 技術 |
|------|------|
| 結構 | HTML5 |
| 樣式 | CSS3、Bootstrap 5 |
| 圖示 | Font Awesome 6 |
| 互動 | jQuery 4、Bootstrap JS |
| 動畫 | Animate.css、WOW.js |
| 其他 | counterUp2（數字計數）、Random User API（隨機頭像） |

---

## 功能特色

- 響應式設計，支援桌機與行動裝置
- Carousel 四款產品系列輪播 Banner
- 數字計數動畫（進入視窗觸發）
- 研發歷程六階段時間軸
- Roller 卷軸滑入效果
- 對角線圖文並陳排版
- 使用者評論串接 Random User API
- 肌膚諮詢預約表單（含 Google Maps）
- 會員登入／註冊 Modal 彈窗
- 回頂部按鈕

## 如何執行

本專案為純靜態網站，無需安裝任何套件。

用瀏覽器直接開啟 `index.html` 即可，建議搭配 VS Code 的 Live Server 擴充套件執行。
